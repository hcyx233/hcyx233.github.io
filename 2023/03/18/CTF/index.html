# 图片隐写术

图片隐写的常用隐写方法：

细微的颜色差别、GIF 图多帧隐藏、exif 信息隐藏、图片修复、最低有效位 LSB 隐写、图片加密

  1. Firework
  2. exif
    1. 查看图片的 exif 信息（地理位置等等）
  3. stegsolve
    1. 使用场景：两张图片信息基本相同
    2. stegsolve 图片隐写解析器的使用_黑朱雀的博客 - CSDN 博客_stegsolve1.3 by caesum
  4. LSB（最低有效位 Least Significant Bit）
    1. 工具：stegsolve，zsteg，wbStego43open
  5. TweakPNG
    1. 使用场景：文件头正常却无法打开文件，利用 TweakPNG 修改 CRC
    2. 有时 CRC 没有错误，但是图片的高度或者宽度发生了错误，需要通过 CRC 计算出正确的高度或者宽度
  6. Bftools
    1. 使用场景：在 Windows 的 cmd 下，对加密过的图片文件进行解密
    2. bftools decode braincopter 要解密的图片名称 --output 输出文件名
    3. bftools run 上一步输出的文件
  7. SilentEye
  8. JPG 图片加密
    1. stegdetect 主要用于分析 jpeg 文件
      1. stegdetect xxx.jpg
      2. stegdetect -s 敏感度 xxx.jpg
  9. outguess
  10. F5
  11. 二维码处理
    1. 电脑桌面二维码扫描器

  2. 过滤端口
    1. tcp.port eq xx
    2. tcp.dstport == 80
  3. 过滤协议
  4. 过滤 MAC
  5. 包长度过滤
  6. http 模式过滤
    1. http.request.method == "GET"
    2. http.request.uri == "path"
    3. http contains "xxx"

# USB 鼠标流量抓取分析

鼠标流量包的数据长度为 4 个字节，第一个字节表示按键，0x00 表示没有按键，0x01 表示左键，0x02 表示右键，第二个字节代表左右偏移，第三个字节代表上下偏移
