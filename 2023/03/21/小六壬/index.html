# 小六壬

# 数字法起卦方式

  1. 万事大安起
  2. 数字法出现 0 的情况，遇见 0 直接忽略，例如 17:05，直接 175
  3. 纯 0 的情况，例如 00:00，直接按空亡来算
  4. 数字突然变化的情况，这是在确认时间的瞬间变化才算
  5. 同声相应，同气相求

# 实战教学

  1. 解卦前提
    1. 具像化问题
    2. 求测者目的
  2. 解卦的核心思路
    1. 不要纠结于断语
    2. 卦象的特点才是最关键的
    3. 不要纠结于五行
    4. 以卦象特点解卦最为直观
  3. 六宫的特点
    1. 六宫之。大安
      1. 静态、稳定发展、不动的一切
      2. 思维引导：正在发生、未发生 (难成)
    2. 六宫之。留连
      1. 延迟、慢动态、输出
    3. 六宫之。速喜
      1. 快速
    4. 六宫之。赤口
      1. 与嘴巴有关的一切、口舌是非
      2. 冲突、聊天
    5. 六宫之。小吉
      1. 佛系、随缘、恰到好处
    6. 六宫之。空亡
      1. 空无、没有、空间
      2. 天空、画大饼
